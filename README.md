# Number Guessing Game 🎯

This is a simple number guessing game made using Python. 
In this game, the user has to guess a randomly generated number.

## Features
- Random number is generated every time
- Gives hints like too high or too low
- Easy to play in terminal

## How to Run
1. Install Python
2. Run the file:
   python number_guessing_game.py
